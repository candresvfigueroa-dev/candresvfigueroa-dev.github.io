# Responde Conmigo 🎮

Juego educativo de habilidades sociales.

## Estructura de carpetas

```
responde-conmigo/
├── index.html
├── estilos.css
├── juego.js
├── README.md
└── img/
    ├── responde_conmigo.png   ← imagen pantalla inicio
    ├── lograste.png           ← imagen pantalla final
    ├── llegas_salon.png       ← nivel 1
    ├── lapiz.png              ← nivel 2
    ├── contando.png           ← nivel 3
    ├── grupo.png              ← nivel 4
    ├── videojuegos.png        ← nivel 5
    ├── fila.png               ← nivel 6
    ├── triste.png             ← nivel 7
    ├── hablando.png           ← nivel 8
    ├── ayuda.png              ← nivel 9
    └── problema.png           ← nivel 10
```

## Cómo abrir el juego

1. Abre la carpeta en VS Code
2. Clic derecho sobre index.html
3. Selecciona "Open with Live Server"

## Música

La música se genera automáticamente con Web Audio API.
No necesitas ningún archivo .mp3 externo.
Presiona el botón 🎵 en la esquina superior derecha para activarla.

## Imágenes

Agrega tus imágenes en la carpeta img/ con los nombres exactos indicados arriba.
Formatos recomendados: .png o .jpg — tamaño sugerido: 600x300 px.
